# AI 原型开发项目

使用 AI 辅助进行移动端原型设计和小程序开发。

## 技术栈

- **AI 模型**: Claude Code + MiniMax 2.1
- **开发工具**: Claude Code CLI
- **框架**: Tailwind CSS（通过 CDN 引入）

## 项目结构

```
projects/
└── example/
    └── short-video-app.html    # 短视频 App 移动端原型
```

## 已完成

- 短视频 App 移动端原型（3 个页面）
  - 首页：短视频信息流，支持上下滑动切换
  - 发现：搜索和分类浏览
  - 个人中心：用户资料和作品展示

## 开发方式

通过 Claude Code CLI 与 AI 模型对话，实现：
- 代码生成和修改
- 前端界面开发
- 样式优化
- Bug 修复
